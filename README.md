# folda
Extension for After Effects that lets you automatically structure project folders both inside your AE project and on your file system (Windows/macOS). Just install the .zxp with ZXP Installer, open the panel, and instantly generate clean, professional folder structures for every project.

It's built as a `.zxp` extension, making installation as easy as dragging the file into **ZXP Installer**.  
Perfect for motion designers, editors, and teams who want a faster, cleaner workflow.
